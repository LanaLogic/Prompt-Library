# 📚 Prompt Library

Библиотека промптов для работы с задачами в области маркетинговых исследований, анализа тендеров и операционного планирования.

---

## 🧠 О проекте

Эта библиотека создана как система для:

* хранения и структурирования промптов
* быстрого поиска и повторного использования
* стандартизации работы с ИИ
* анализа тендерной документации
* принятия управленческих решений

---

## 🗂 Структура

```text
Prompts/     — готовые промпты
Templates/   — шаблоны для создания новых промптов
Archive/     — устаревшие или неактуальные промпты
Examples/    — примеры использования (если есть)
```

---

## 🚀 Как использовать

### 1. Выбери нужный промпт

Перейди в папку:

```
Prompts/
```

---

### 2. Открой файл

Каждый файл = один промпт

---

### 3. Подставь данные

Замени переменные:

```text
{{input_data}}
{{tender_docs}}
{{audience}}
```

---

### 4. Скопируй и вставь в ИИ

Используй в ChatGPT или другом инструменте.

---

## 🔄 Workflow (рекомендуемый)

```text
Документы → Выжимка → Анализ → Решение
```

### Шаг 1 — Выжимка

Извлечение ключевой информации из тендера

### Шаг 2 — Анализ

Оценка:

* сложности
* рисков
* ресурсов
* экономики

### Шаг 3 — Решение

* участвовать / не участвовать
* аргументация

---

## 🧾 Формат промптов

Каждый промпт содержит:

* Роль
* Задачу
* Контекст
* Входные данные
* Логику
* Формат ответа
* Ограничения

## 🧩 Properties (метаданные)

Используются для структурирования:

```yaml
type: prompt
status: draft | ready | tested
model: gpt-5
use_case: analysis | writing | etc
rating: 1–5
owner: 
updated: 
```

---

## 🔍 Поиск

Можно искать по:

* тегам (`#analysis`)
* статусу (`status: ready`)
* задаче (`use_case: analysis`)
